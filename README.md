# UKiS 



ツールの説明

本プログラムは、ヒト遺伝子のイントロンの長さ分布をボックスプロットで描画するためのツールである。

Refseqを元に解析を行う。

特徴としては、自分の興味の対象となる遺伝子が、分布のどこに位置するのかを図示することができる。
